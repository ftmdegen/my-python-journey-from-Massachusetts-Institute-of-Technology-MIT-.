##  Learning Approach

1. Understand the problem - Write pseudocode/recipe
2. Break it down - Identify steps needed
3. Code incrementally - Build feature by feature
4. Test thoroughly - Try different inputs
5. Document - Add comments and explanations
6. All about first principles thinking
